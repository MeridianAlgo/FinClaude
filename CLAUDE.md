# Financial Analysis Skills Suite

This project is a comprehensive collection of Claude Code skills for in depth financial analysis, modeling, and decision making.

## Project Structure

All skills live under `.claude/skills/` and are organized by financial domain:

- `financial-analysis` : Deep fundamental analysis of companies (income statements, balance sheets, cash flows)
- `dcf-valuation` : Discounted Cash Flow modeling with sensitivity analysis
- `ratio-analysis` : Comprehensive financial ratio computation and benchmarking
- `portfolio-optimizer` : Modern Portfolio Theory based optimization and risk analysis
- `options-pricer` : Black Scholes and binomial options pricing with Greeks
- `technical-analysis` : Chart pattern recognition, indicator computation, and signals
- `risk-assessment` : VaR, CVaR, stress testing, and scenario analysis
- `earnings-analyzer` : Earnings call transcript analysis and surprise detection
- `credit-analysis` : Credit risk scoring, Altman Z score, and default probability
- `macro-dashboard` : Macroeconomic indicator tracking and analysis
- `financial-report` : Generate comprehensive PDF style financial reports
- `bond-analysis` : Fixed income analysis — yield/duration/convexity, credit spreads, relative value
- `dividend-analysis` : Dividend sustainability scoring, safety rating, DDM valuation, and growth projection
- `ipo-analysis` : IPO prospectus analysis, valuation vs. comps, lockup risk, and participation verdict
- `m-and-a-analysis` : M&A deal analysis — synergies, accretion/dilution, deal structure, and regulatory risk

## Conventions

- All monetary values should use USD unless otherwise specified
- Percentages should be displayed to 2 decimal places
- Large numbers should use comma separation (e.g., 1,000,000)
- All analysis should include data sources and timestamps
- Risk metrics should include confidence intervals
- Never provide investment advice; present analysis objectively
- Always include disclaimers when presenting financial analysis
